Acceso a la API de Ferre+ (Solo para Administradores)

Para probar nuestra API, necesitas un Access Token que tiene una validez de un día. Este token se utiliza para autenticar y autorizar tus solicitudes a la API.

Pasos para obtener y utilizar el Access Token:

  1. Solicita un Access Token a través del punto final de autenticación.
  2. Usa el Access Token en el encabezado de autorización de tus solicitudes API.
  3. El token expira después de 24 horas, momento en el cual necesitarás obtener uno nuevo.
